
## 📋 简历项目描述

### 陆地旅行智能推荐系统 (Land Travel Smart Recommender)

**技术栈：** Vue3 + Vite + Vant4 · FastAPI (Python) · DeepSeek LLM API · 高德地图 API · 和风天气 API · Supabase · PostgreSQL

**项目简介：** 单人可运维的轻量化智能旅行路书工具，集成 AI 行程生成、多轮问答、实时天气、全国370个城市数据，采用小米风格 UI 设计。

**核心亮点：**

1. **AI Agent 独立服务** — 拆分为独立部署的 AI Agent 服务（FastAPI，端口8001），通过 HTTP 与主后端解耦通信。内置轻量 ToolCall 框架（抽象基类 + 注册表 + 调度器），注册了 3 个工具（天气查询/POI 状态/路书知识库），LLM 通过 Function Calling 实时获取数据生成个性化行程，最多 3 轮工具调用循环，杜绝无限耗损 API 额度。

2. **LLM 异步调用与输出校验** — 基于 DeepSeek Chat API，全程异步非阻塞。双层输出校验：第一层 JSON 格式自动修复（补逗号/括号截断/代码块提取），第二层内容合理性校验（必填字段/景点时间顺序/违禁词筛查），异常输出自动降级为静态路书。

3. **多轮问答上下文管理** — 内存短时上下文缓存（TTL 10 分钟，最多 20 条消息），支持连续对话，Session ID 驱动上下文隔离。独立于主服务的成本风控计数器。

4. **全国城市数据覆盖** — 通过高德地图行政区域 API 获取全国 370 个地级市数据，自动绑定 Wikipedia/Flickr 真实城市图片，支持标签筛选（文化/美食/亲子/自驾/自然）和搜索。

5. **异步高并发架构** — FastAPI 全程异步，LLM 调用不阻塞主线程；预扣式风控系统（日预算硬上限、游客/用户分离限流），彻底杜绝 API 费用失控。

6. **全链路容错降级** — 三级数据兜底策略（DB→LRU缓存→JSON文件），任一外部服务故障不影响核心功能，天气/地图/AI 均支持 N/A 降级。

7. **小米风格 UI** — Vue3 + Vant4 移动端 SPA，自适应桌面布局，橙色品牌色系，产品卡片式网格展示。

8. **自动运维体系** — 一键启动脚本 (Python/bat)，进程存活监控，健康检查端点，进程异常自动恢复。

**技术细节：**
- 前端：Vue3 Composition API + Pinia 状态管理 + Vue Router + Axios
- 后端：FastAPI + Pydantic + httpx 异步 HTTP 客户端
- **AI Agent 独立服务**：FastAPI 独立部署，DeepSeek Chat API + Function Calling，ToolCall 框架（注册表/调度器/执行器），3 轮工具循环限制
- Agent 工具：天气查询（和风天气）、POI 状态查询、路书知识库读取，通过 BackendProxy 反查主项目数据
- 输出校验：双层校验器（JSON 自动修复 + 内容合理性审核），异常降级为静态路书
- 上下文管理：内存缓存（10 分钟 TTL，最多 20 条），Session ID 隔离，过期自动清理
- 成本风控：预扣式日预算 30 元硬上限 + 游客日限 5 次 + 登录用户日限 10 次
- 数据：内存 LRU 缓存 + 和风天气 API + 高德地图 API + JSON 文件存储
- 部署：双服务（主后端:8000 + AI Agent:8001），Nginx 反向代理
